# Zenn コンテンツ管理リポジトリ

Zenn の記事と本を管理するためのリポジトリです。

## 使い方

### 記事作成

```bash
npm run new:article
```

### 本の作成

```bash
npm run new:book
```

### プレビュー

```bash
npm run preview
```

ブラウザで `http://localhost:8000` を開くとプレビューを確認できます。
